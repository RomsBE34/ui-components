Le Wagon's components

Getting started

Add a new CSS file (e.g. avatar.css) in your components, copy/paste our CSS, then use the HTML code of the components wherever you want. Here are available components:

  • Avatars
  • Buttons
  • Banners
  • Cards
  • Messages
  • Navbar
  • Footer
  • Tabs
  • List with upvote
